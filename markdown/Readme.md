## Code

### Inline Code

You can print to the terminal using the `puts "hello world"` command.

**`puts "hello world"`**

### Multi line Code

#### Without highlighting

```
def hello_world
    puts "Hello World"
end
```

#### With highlighting

```py
for i in range(1,5):
    print(a[i])
```
